Head

Voltar

O elemento head é um contêiner para metadados (dados sobre dados) e é colocado entre a tag html e a tag body.
Metadados são dados sobre o documento HTML. Os metadados não são exibidos.
Os metadados normalmente definem o título do documento, conjunto de caracteres, estilos, scripts e outras metainformações.
Os seguintes elementos podem estar dentro do elemento head:
title (obrigatório em todo documento HTML)
style
base
link
meta
script
noscript

Title

Voltar

A tag title define o título do documento. O título deve ser somente texto e é mostrado na barra de título do navegador ou na aba da página.
A tag title é obrigatória em documentos HTML!
O conteúdo do título de uma página é muito importante para a otimização de mecanismos de pesquisa (SEO)! O título da página é usado pelos algoritmos do mecanismo de pesquisa para decidir a ordem ao listar as páginas nos resultados da pesquisa.
O elemento title:
define um título na barra de ferramentas do navegador
fornece um título para a página quando ela é adicionada aos favoritos
exibe um título para a página nos resultados do mecanismo de pesquisa
Aqui estão algumas dicas para criar bons títulos:
Escolha um título mais longo e descritivo (evite títulos de uma ou duas palavras)
Os mecanismos de pesquisa exibirão cerca de 50 a 60 caracteres do título, então tente não ter títulos maiores que isso
Não use apenas uma lista de palavras como título (isso pode reduzir a posição da página nos resultados de pesquisa)
Portanto, tente tornar o título o mais preciso e significativo possível!
Nota: Você NÃO pode ter mais de um elemento title em um documento HTML.

Body

Voltar

A tag body define o corpo do documento.
O elemento body contém todo o conteúdo de um documento HTML, como títulos, parágrafos, imagens, hiperlinks, tabelas, listas, etc.
Nota: Só pode haver um elemento body em um documento HTML.

H1 - H6

Voltar

As tags h1 a h6 são usadas para definir cabeçalhos HTML.
h1 define o título mais importante. h6 define o título menos importante.
Nota: Use apenas um h1 por página isso deve representar o título/assunto principal de toda a página. Além disso, não pule os níveis de título comece com h1, depois use h2 e assim por diante.

P

Voltar

A tag p define um parágrafo.
Os navegadores adicionam automaticamente uma única linha em branco antes e depois de cada elemento p.
Dica: Use CSS para estilizar parágrafos.

Mark

Voltar

A tag mark define o texto que deve ser marcado ou destacado.

Small

Voltar

A tag small> define textos menores (como direitos autorais e outros comentários secundários).
Dica: Esta tag não está obsoleta, mas é possível obter um efeito mais rico (ou igual) com CSS.

I

Voltar

A tag i define uma parte do texto com uma voz ou humor alternativo. O conteúdo interno normalmente é exibido em itálico.
A tag i é frequentemente usada para indicar um termo técnico, uma frase de outro idioma, um pensamento, o nome de um navio, etc.
Utilize o elemento i somente quando não houver um elemento semântico mais apropriado, como:
em (texto enfatizado)
strong (texto importante)
mark (texto marcado/destacado)
cite (o título de uma obra)
dfn (um termo de definição)

U

Voltar

A tag representa algum texto desarticulado e com estilo diferente do texto normal, como palavras com erros ortográficos ou nomes próprios em texto chinês. O conteúdo interno normalmente é exibido com um sublinhado. Você pode alterar isso com CSS (veja o exemplo abaixo).
Dica: Evite usar o elemento u onde possa ser confundido com um hiperlink!

Strong

Voltar

A tag strong é usada para definir textos de grande importância. O conteúdo interno normalmente é exibido em negrito.
Dica: Use a tag b para especificar texto em negrito sem qualquer importância extra!

Ol

Voltar

A tag ol define uma lista ordenada. Uma lista ordenada pode ser numérica ou alfabética.
A tag li é usada para definir cada item da lista.
Dica: Use CSS para estilizar listas.
Dica: Para lista não ordenada, use a tag ul.

Ul

Voltar

A tag ul define uma lista não ordenada (com marcadores).
Use a tag ul junto com a tag li para criar listas não ordenadas.
Dica: Use CSS para estilizar listas.
Dica: Para listas ordenadas, use a tag ol.

Li

Voltar

A tag li define um item da lista.
A tag li é usada dentro de listas ordenadas (ol), listas não ordenadas (ul) e em listas de menu (menu).
Em ul e menu, os itens da lista geralmente serão exibidos com marcadores.
Em ol, os itens da lista geralmente serão exibidos com números ou letras.
Dica: Use CSS para estilizar listas.

A

Voltar

A tag a define um hiperlink, que é usado para vincular de uma página a outra.
O atributo mais importante do elemento a é o atributo href, que indica o destino do link.
Por padrão, os links aparecerão da seguinte forma em todos os navegadores:
Um link não visitado está sublinhado e azul
Um link visitado está sublinhado e roxo
Um link ativo está sublinhado e vermelho

Hr

Voltar

A tag hr define uma quebra temática em uma página HTML (por exemplo, uma mudança de tópico).
O elemento hr é mais frequentemente exibido como uma regra horizontal usada para separar o conteúdo (ou definir uma alteração) em uma página HTML.

Sub

Voltar

A tag sub define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H2O.
Dica: Use a tag sup para definir texto sobrescrito.

Sup

Voltar

A tag sup define texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e às vezes é renderizado em uma fonte menor. Texto sobrescrito pode ser usado para notas de rodapé, como WWW.
Dica: Use a tag sub para definir o texto subscrito.

Blockquote

Voltar

A tag blockquote especifica uma seção citada de outra fonte.
